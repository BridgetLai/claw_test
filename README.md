# claw_test

Test repository for OpenClaw experiments and development.

## Purpose

This repository serves as a testing ground for OpenClaw workflows, skills development, and automation experiments.

## Features

- OpenClaw integration testing
- Skill development examples
- Automation workflows
- Project templates

## Contents

- `snake-game.html`: Classic Snake game in HTML5 Canvas
- `snake-game.md`: Snake game documentation
- `docs/`: Documentation directory
- `scripts/`: Example scripts

## How to Access

The snake game can be accessed locally by opening `snake-game.html` in a web browser.
