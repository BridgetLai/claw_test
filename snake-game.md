# Snake Game

A classic Snake game implemented in HTML5 Canvas.

## Features

- Classic snake gameplay with arrow controls
- Score tracking and high score system
- Speed increases as you eat more food
- Game states: Start, Pause, Reset
- Responsive design

## How to Play

1. Click "Start Game" button or press arrow keys
2. Use arrow keys (↑, ↓, ←, →) to control snake direction
3. Eat the red food to grow and increase score
4. Avoid hitting walls or yourself
5. Press 'P' to pause, 'R' to reset

## Files

- `snake-game.html`: The main game file
- `README.md`: Project overview

## Deployment

This game can be deployed to GitHub Pages or run locally by opening the HTML file in a browser.
